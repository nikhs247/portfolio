## Project Overview
The Armada framework required elastic resource management to handle heterogeneous edge-dense environments efficiently. 

## Technical Implementation
* **Resource Allocation:** Implemented core resource allocation logic using GoLang.
* **Server Selection:** Created a latency-aware server selection algorithm utilizing gRPC.
* **Impact:** The resulting architecture achieved an 85% reduction in end-to-end latency.
