## Project Overview
Meeting federal security standards requires a comprehensive approach to both software pipelines and hardware environments. This work focused on establishing a CMMC-compliant CI/CD pipeline and hardening local infrastructure.

## Technical Implementation
* **Pipeline Architecture:** Deployed a GitLab CI/CD pipeline on local infrastructure to enforce strict, secure development lifecycles.
* **Hardware Compliance:** Configured and hardened local hardware to be fully CMMC 2.0 compliant, ensuring that physical access points meet the same rigorous standards as the cloud infrastructure.
